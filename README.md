# Frontend Docs

Краткая база знаний по фронтенду: браузер, HTML, CSS, JavaScript, React и прикладные инструменты.

Это не официальная документация и не полноценный учебник. Репозиторий собран как компактный набор заметок и конспектов, который помогает быстро повторять ключевые темы и держать важные практические вещи в одном месте.

Материал основан на документации, статьях, учебных материалах и личных заметках, а затем переписан в более коротком и прикладном формате.

## Что внутри

- [Browser](./Browser.md) — как работает браузер: DNS, TCP, TLS, HTTP, рендеринг, пайплайн загрузки страницы
- [Baseline](./Baseline.md) — хронологический список новых web-platform фич по годам с короткими заметками и ссылками
- [HTML and CSS](./HTML%20and%20CSS.md) — семантика HTML, формы, метатеги, CSS-база, layout, анимации, flex, grid и другое
- [JS part 1](./JS%20part%201.md) — базовые темы JavaScript
- [JS part 2](./JS%20part%202.md) — продолжение JavaScript и более сложные темы
- [React](./React.md) — конспект по React
- [Axios](./Axios.md) — краткий конспект по запросам, конфигу, interceptors, отмене запросов, ошибкам и мокированию

## Важное замечание

Часть формулировок намеренно упрощена, чтобы материал было легче быстро повторять и использовать как быстрый reference.
Если нужна максимальная точность по конкретной теме, лучше дополнительно сверяться с официальной документацией и спецификациями.

## Назначение репозитория

Репозиторий создан как компактная база знаний для быстрого повторения ключевых frontend-тем, онбординга и повседневной работы.

Если репозиторий оказался полезным, можно поставить `star`.
